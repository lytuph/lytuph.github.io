JSON概念

JSON(JavaScript Object Notation, JS 对象标注) 是一种轻量级的数据交换格式。采用完全独立于编程语言的文本格式来存储和表示数据。简洁和清晰的层次结构使得 JSON 成为理想的数据交换语言。 易于人阅读和编写，同时也易于机器解析和生成，并有效地提升网络传输效率。

当客户端和服务器传输的信息（参数信息和响应信息）简单时，可采取一般的文本格式，当传输的数据量很大且结构复杂时，可以考虑使用结构化的字符串进行数据传输，如xml或json等，在Ajax技术中，一般使用json实现数据传输。

JSON语法是javaScript语法的子集，javaScript用[]中括号来表示数组，用{}大括号来表示对象。

Json包括两种基本结构：JSON对象和JSON数组，语法规则:

  1. 数据为 键/值 对。
  2. 数据由逗号分隔。
  3. 大括号保存对象
  4. 方括号保存数组。
  5. JSON对象：保存多个键值对。 {"name":"Runoob", "url":"www.runoob.com"}

JSON数组

数组表示为“[]”扩起来的内容，格式为["java","javascript","vb",...]，通过“数组名.[index]”获取数组元素，元素类型可以是数值、字符串、数组、对象等。

xxxxxxxxxx
[   {"name":"Runoob", "url":"www.runoob.com"},
    {"name":"Google", "url":"www.google.com"},
    {"name":"Taobao", "url":"www.taobao.com"}
]

xxxxxxxxxx

    "age":62,
    "Cars":[
        { "car":"porsche", "model":"911" },
        { "car":"BMW", "model":"M5" },
        { "car":"Volvo", "model":"XC90" }
        ]
}

Js在与后面交换数据时，一般使用文本串，或Json。文本串适合传输少量不复杂的数据，使用split()拆开；

Json是js原生态支持的，相对xml来说，更小，更快；它可以让大量的数据结构化，访问里面关键字，如同访问对象一样方便，所以专业的学生必须学会json最简单的用法。

Json本身还是比较复杂，比如它与文件之间的打交道，很复杂的变形，在我们一般学习时暂时没有必要。特别是AJAX异步技术与JSON结合，前后台传数据非常方便

json对象

对象表示为“{}”扩起来的内容，格式为{key1:value1,key2:value2,...}的键值对集合。通过“对象.key”获取属性值，属性值类型可以是数值、字符串、数组、对象等。

例：下面看一个简单的JSON对象： 关键字和值要用双引号，不能使用单引号

xxxxxxxxxx
function Json1()
 {  var obj={
       name:"张明明",
       age:23,
       pw:"111111",
       intru:"我是一个兵"
     }
   alert(obj.name);alert(obj.intru); return obj.age;
}


<input type="button" onclick="window.alert(Json1());"/>

json数据的解析（反序列化）

一般使用eval()函数来解析json中文本数据到一个JS对象，然后在JS中随意操作了（即使不解析也可以直接操作JSON）

什么场合非要用到解析呢？JSON经常应用到的场景是：在后台应用程序中将相应数据封装成JSON格式，传到前台页面之后，需要将JSON格式转换为JavaScript对象，然后在网页中使用该数据。 例：

xxxxxxxxxx

{
  var stud=[

   { name:"张明亮",   age:22,   pw:"111111", intru:"学习努力"}
     ];
   var i=2;
   alert(stud[i].name);  //下标从0开始
   for(var j=0;j<stud.length;j++)
   { alert(stud[j].intru);
   }
   var aa=eval(stud);
   alert(aa[3].name);
   //为什么非要JSON解析呢？ 应该场合主要是从WEB服务器java/c#服务器程序中读出数据库中的数据后，一般是以字符串形式或json形式传回来的。你要么使用split去拆开，要么把json格式的串解析成js对象
   //我的理解，js对象应该是简化版的java的arraylist类（线性表）
